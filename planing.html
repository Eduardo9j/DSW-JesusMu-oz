Tema de estudio

 

 

Contenido en HTML ( SECTION, ASIDE, FOOTER,ARTICLE)

 

 

Objetivos

 

 

Objetivo principal

 

El objetivo principal es el de entender por medio de medios didacticos y tutoriales el uso de las etiquetas de html SECTION, ASIDE, FOOTER,ARTICLE.

 

Objetivos especificos

 

  • Investigar acerca de las funciones de cada etiqueta
  • Explicar teoricamenente que hace cada una de las etiquetas
  • Realizar video tutoriales acerca del uso de las etiquetas

    •  

     

    Contenido

     

     

    Bienvenidos al curso de diseño de sitios web, en esta actividad se dará una tematica acerca de la materia haciendo una consulta sobre algunos elementos y etiquetas que componen el diseño web en HTML. En el siguiente trabajo practico se quieren conocer algunos de los elementos que contiene HTML 5 asÍ como también diseñar una página donde se aloje todo el contenido a desarrollar. En esta oportunidad definiremos los conceptos referentes al tema elegido, el cual fue Contenido en HTML ( SECTION, ASIDE, FOOTER, ARTICLE).
